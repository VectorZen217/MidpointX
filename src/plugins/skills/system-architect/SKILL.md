# System Architect

This skill provides a senior-level architectural workbench.

## Workflows

### 1. Architectural Audit
Use `scripts/map_dependencies.cjs` to understand the current structure before proposing changes.

### 2. Design & Visualization
Use `scripts/draw_mermaid.cjs` to generate C4 or sequence diagrams for new designs.

### 3. Migration Planning
Consult `references/migrations/` for step-by-step blueprints.

### 4. Self-Improvement
All approved designs are logged via `scripts/log_pattern.cjs` to `.architect/memory/SIGNALS.json`.
