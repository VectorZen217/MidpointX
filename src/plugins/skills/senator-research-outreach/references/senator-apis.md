# US Senator APIs

To access voting records, legislative actions, and campaign finance data, use these APIs:

## 1. ProPublica Congress API
- **Endpoint:** `https://api.propublica.org/congress/v1/`
- **Data:** Roll-call votes, member profiles, bill details, committee assignments.
- **Update Frequency:** Every 30 minutes for roll-call votes.
- **Authentication:** Requires an API key (X-API-Key header).
- **Docs:** [ProPublica API Documentation](https://projects.propublica.org/api-docs/congress-api/)

## 2. Congress.gov API
- **Endpoint:** `https://api.congress.gov/v3/`
- **Data:** Official source for bill text, summaries, and legislative actions.
- **Authentication:** Requires an API key.
- **Docs:** [Congress.gov API Documentation](https://api.congress.gov/)

## 3. OpenFEC API
- **Endpoint:** `https://api.openfec.gov/v1/`
- **Data:** Campaign finance data, fundraising, and expenditures.
- **Authentication:** Requires an API key.
- **Docs:** [OpenFEC API Documentation](https://api.openfec.gov/developers/)
