# Logic Shift: THEOREM_SYS_01
Trace ID: TELEGRAM-1781207082311
Learned At: 2026-06-11T19:45:15.989Z

## Justification
The standard approach of gathering information that is already provided in the system prompt or context adds unnecessary steps and processing time. Codifying this direct retrieval method optimizes for efficiency when the information is readily available.

## Discovered Pattern
Listing core capabilities and available skills for operator awareness.

## Optimized Approach
When asked to list capabilities or skills, directly present the information from the system prompt and tool output. Avoid unnecessary intermediate steps or tool calls if the information is already present in the context.
