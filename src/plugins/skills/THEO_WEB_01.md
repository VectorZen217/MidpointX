# Logic Shift: THEO_WEB_01
Trace ID: TELEGRAM-1778777848219
Learned At: 2026-05-14T16:57:53.882Z

## Justification
The standard approach of direct web scraping failed due to robots.txt restrictions. This theorem codifies a fallback strategy that prioritizes user assistance and alternative data acquisition methods when direct programmatic access is not feasible, ensuring the task can still be completed with user collaboration.

## Discovered Pattern
Web search for specific items results in robots.txt blocking programmatic access.

## Optimized Approach
When direct web scraping is blocked by robots.txt, pivot to a strategy that involves informing the user of the limitation and offering alternative methods, such as manual searching or direct dealership website visits. If the user agrees, the agent can then assist with finding dealership websites or formatting manually gathered information.
