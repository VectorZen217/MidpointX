# Data Science & Visualization Lab

This skill provides a senior-level data science environment.

## Workflows

### 1. Data Profiling & Cleaning
Use `scripts/clean_data.py` to identify and fix quality issues in your datasets.

### 2. Statistical Modeling
Implement regression, classification, or clustering models using the modeling suite.

### 3. High-Fidelity Visualization
Use `scripts/generate_viz.py` to create static or interactive charts.

### 4. Self-Improvement
All approved chart styles and model preferences are logged via `scripts/log_insights.cjs` to `.datalab/memory/PREFERENCES.json`.
