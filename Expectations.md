When using MidpointX, you should expect a high-autonomy "Sovereign Gateway" experience that behaves more like a human operator than a simple command script.

Here is what defines the operational experience:

1. Visual Grounding ("Eyes")
Unlike traditional agents that rely solely on DOM or API data, MidpointX sees your screen.

What to expect: The agent will take snapshots to verify your environment, locate buttons visually, and confirm that an action (like a click) actually changed the UI.
Transparency: It will explicitly tell you what it sees (e.g., "I see the browser is open to the login page").
2. Mechanical Execution ("Hands")
The agent is equipped with native desktop automation (Mouse, Keyboard, Browser) and filesystem access.

What to expect: You can ask it to perform tasks across any application, not just web browsers. It can move the mouse, type text, and manipulate files in authorized directories (like D:\Reports).
Policy Guarding: It operates with "Lead Shielding" (a deterministic Policy Engine) that blocks dangerous system commands or unauthorized path access.
3. Proactive Self-Evolution ("Brain")
MidpointX is designed to learn from every mission.

What to expect: If the agent discovers a superior way to solve a task, it will propose a Logic Shift (Theorem). Once approved, this becomes a permanent Skill in its library, making it faster and more reliable in future sessions.
Proactivity: It doesn't just wait for you to type; it can monitor webhooks or file changes and evaluate if it should take action autonomously.
4. Operational Directives
Over-Communication: The agent will explain its reasoning before every tool call.
Temporal Verification: After a UI interaction, it performs a "Motion Probe" (burst capture) to confirm the system state actually transitioned.
Anti-Laziness: It is instructed never to ask you to "manually finish" a task that it has the tools to complete itself.
Summary of Expectations
Feature	Expectation
Reliability	High, due to visual verification and re-planning logic.
Speed	"Human-speed" for visual tasks; high-speed for API tasks.
Security	Local-first, with strict policy-based guarding of core OS files.
Interaction	Collaborative—it works alongside you, keeping you updated on its "heartbeat."
In short, expect an assistant that is self-aware, visually grounded, and constantly improving its own technical library based on your specific workflow.
